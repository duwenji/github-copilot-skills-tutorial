# PUBLISHING

## Pre-publish flow
1. `./.github/skills-config/ebook-build/invoke-build.ps1` を実行する
2. 以下の成果物を確認する
   - `ebook-output/github-copilot-skills-tutorial.epub`
   - `ebook-output/github-copilot-skills-tutorial.pdf`
   - `ebook-output/github-copilot-skills-tutorial-kdp-registration.md`
3. `README.md` / `docs/index.md` / `MASTER-INDEX.md` の導線を確認する
4. サンプル・Skill 記述例・KDP 登録情報が現行内容と合っているか確認する
5. GitHub Pages 表示を確認して公開する
