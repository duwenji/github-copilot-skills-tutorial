# PUBLISHING

## Pre-publish flow
1. `./.github/skills-config/ebook-build/invoke-build.ps1` を実行する
2. `ebook-output/github-copilot-skills-tutorial.epub` を確認する
3. `README.md` / `docs/index.md` / `MASTER-INDEX.md` の導線を確認する
4. サンプルや Skill 記述例が現行の VS Code / Copilot 仕様に合っているか確認する
5. GitHub Pages 表示を確認して公開する
